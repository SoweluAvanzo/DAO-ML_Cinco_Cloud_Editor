## Description

The framework application that runs the cloud instance of Cinco. Takes care of things like browser GUI, user account management and project management. 

## Used Technologies

[Docker][docker] - Facilitating set up and development.

[Angular/TS][angular] - Frontend language.

[Java][java] - Backend language.

[Quarkus][quarkus] - Backend framework
